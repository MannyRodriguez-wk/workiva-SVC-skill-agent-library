# Safety (read once)

Optional. Only matters if you're using skills on customer work, internal apps, or sensitive data.

---

## When using skills in Claude

- Say **yes** when Claude asks to read skill files from your machine — that's normal
- Say **no** if Claude asks for access you don't understand — ask a teammate first
- Don't paste **passwords, API keys, or customer private info** into chat

## When using the internal-apps skill (Google Sheets / Confluence tools)

- Deploy apps as **Workiva-only**, never public
- Anyone with the link can **read** what's in the app — don't store HR, comp, or secret customer data there
- Help channel: **#cop-ai-internal-apps**

## Before you send anything to a customer

- A person on your team should read it first
- Check that facts and numbers are real — Claude can sound confident and be wrong

That's the whole list.
